## 📋 **Complete List of Usable Functions in EduSecure**

### **🎥 Core Monitoring Functions**

#### **1. Real-Time Video Capture**
- **Function**: Opens webcam and displays live video feed
- **Usage**: `py -3.11 main.py`
- **Features**:
  - 640x480 resolution
  - Real-time display in "EduSecure" window
  - Press 'q' to quit

#### **2. Face Detection**
- **Function**: Detects faces in video stream using OpenCV Haar cascades
- **Usage**: Automatic in main.py
- **Features**:
  - Multi-scale detection
  - Frontal face detection
  - Real-time processing

#### **3. Attendance Tracking**
- **Function**: Automatically marks attendance when faces are detected
- **Usage**: Automatic in main.py
- **Features**:
  - Logs to SQLite database
  - Records date, time, and status
  - Currently uses "Demo Student" placeholder
  - Prevents duplicate entries

### **💾 Database Functions**

#### **4. Database Initialization**
- **Function**: Creates SQLite database with required tables
- **Tables Created**:
  - `students` - Student information and face encodings
  - `attendance` - Attendance records
  - `incidents` - Behavioral incident logs

#### **5. Data Storage**
- **Function**: Persistent storage of all monitoring data
- **Features**:
  - SQLite database (edusecure.db)
  - Automatic table creation
  - Thread-safe operations

### **📧 Communication Functions**

#### **6. Email Alert System**
- **Function**: Sends email notifications with attachments
- **Configuration**: Gmail SMTP (currently configured)
- **Features**:
  - HTML email format
  - Video clip attachments
  - Configurable sender/receiver
  - TLS encryption

#### **7. Incident Recording**
- **Function**: Logs behavioral incidents to database
- **Features**:
  - Timestamp recording
  - Video clip path storage
  - Student association (when available)

### **🎬 Video Recording Functions**

#### **8. Video Clip Capture**
- **Function**: Records 5-second video clips on demand
- **Features**:
  - XVID compression
  - Automatic filename generation
  - Saves to `video_clips/` directory
  - 20 FPS recording

### **📊 Reporting Functions**

#### **9. Attendance Reports**
- **Function**: Generates CSV reports of attendance data
- **Usage**: `py -3.11 report.py`
- **Output**: `attendance_report_all.csv`
- **Includes**: Student names, dates, times, status

#### **10. Incident Reports**
- **Function**: Generates CSV reports of behavioral incidents
- **Usage**: `py -3.11 report.py`
- **Output**: `incident_report.csv`
- **Includes**: Student names, timestamps, video paths

#### **11. Summary Reports**
- **Function**: Creates text summary of all activities
- **Usage**: `py -3.11 report.py`
- **Output**: `summary_report.txt`
- **Includes**: Total students, incident counts, statistics

### **🔧 Utility Functions**

#### **12. Student Registration (Framework)**
- **Function**: Framework for registering student faces
- **Current Status**: Requires InsightFace installation
- **Features**:
  - Face encoding capture
  - Database storage
  - Individual student identification

#### **13. Advanced Cheating Detection**
- **Function**: Real-time eye movement and head position analysis
- **Current Status**: ✅ ACTIVE - OpenCV-based implementation
- **Features**:
  - Eye detection and gaze tracking
  - Head position analysis (tilt detection)
  - Suspicious behavior identification
  - Visual alerts on screen
  - Automatic video recording on detection
  - Email alerts with evidence

### **🎮 User Interface Functions**

#### **14. Console Output**
- **Function**: Provides real-time feedback
- **Features**:
  - Attendance confirmations
  - System status messages
  - Error reporting

#### **15. Video Display**
- **Function**: OpenCV window management
- **Features**:
  - Real-time video display
  - Keyboard controls (q to quit)
  - Window management

### **⚙️ Configuration Functions**

#### **16. Email Configuration**
- **Function**: Configurable SMTP settings
- **Current Setup**: Gmail with app password
- **Editable**: sender, receiver, server, port

#### **17. Directory Management**
- **Function**: Automatic creation of required folders
- **Creates**: `video_clips/` directory
- **Features**: Error handling for permissions

### **🔄 System Management**

#### **18. Graceful Shutdown**
- **Function**: Proper cleanup on exit
- **Features**:
  - Camera release
  - Window closure
  - Resource cleanup

#### **19. Error Handling**
- **Function**: Robust error management
- **Features**:
  - Camera access checks
  - Database connection handling
  - Import error management

## 🚀 **How to Use Each Function**

### **Start Monitoring:**
```bash
cd Capstone
py -3.11 main.py
```

### **Generate Reports:**
```bash
py -3.11 report.py
```

### **Test Email:**
```bash
py -3.11 -c "from main import send_email; send_email('Test', 'Hello!', None)"
```

### **Check Database:**
- File: `edusecure.db`
- Tables: students, attendance, incidents

### **View Video Clips:**
- Folder: `video_clips/`
- Format: AVI files with timestamps
